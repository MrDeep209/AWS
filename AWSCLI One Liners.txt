## Install SSM Agent
sudo yum install -y https://s3.us-west-2.amazonaws.com/amazon-ssm-region/latest/linux_amd64/amazon-ssm-agent.rpm

## Generate IAM Report
aws iam generate-credential-report
aws iam get-credential-report --output text --query Content | base64 -d >> dev-monthly-user-report.csv

## Install EPEL
sudo amazon-linux-extras install epel -y
